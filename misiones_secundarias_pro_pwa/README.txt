# Misiones Secundarias PRO

## Qué viene en esta carpeta
- `index.html`
- `manifest.json`
- `sw.js`
- carpeta `icons/`

## Cómo dejarla instalable en el celular
Esta versión es una PWA. Para que el botón **Instalar app** aparezca bien, debes subir estos archivos a una web segura (por ejemplo GitHub Pages).

### Opción rápida: GitHub Pages
1. Crea una carpeta o repositorio nuevo.
2. Sube **todos** los archivos de esta carpeta.
3. Activa GitHub Pages.
4. Abre la URL desde Chrome en tu celular.
5. Toca **Instalar app** o usa **Agregar a pantalla de inicio**.

## Nota
Si la abres como archivo local (`file://`), puede funcionar, pero la instalación como app completa depende de que esté publicada en una web.
